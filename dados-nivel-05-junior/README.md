# 🟡 Dados Nível 05: Criação de Dashboard Executivo (Júnior)

## 📜 O Desafio
O CEO da empresa não quer ver tabelas do Excel nem scripts Python; ele quer um painel visual para acompanhar a saúde da empresa diariamente. 
Usando um dataset fictício de Vendas (contendo: Data da Venda, Produto, Categoria, Região, Vendedor e Receita), crie um **Dashboard Interativo**.

O dashboard deve conter:
- KPIs: Receita Total, Número de Vendas, Ticket Médio.
- Um gráfico de linha mostrando a evolução da Receita ao longo do tempo (meses).
- Um gráfico de barras ranqueando as Regiões por volume de vendas.
- Filtros interativos para que o CEO possa filtrar o dashboard inteiro por "Ano" ou "Categoria".

## 🎯 Objetivo
Avaliar proficiência no uso de ferramentas de Business Intelligence e a capacidade de contar uma história através dos dados.

## 🛠️ Regras
- **Ferramenta:** Power BI, Tableau, Metabase, Looker Studio, ou até mesmo um dashboard em Python usando Streamlit / Dash.
- **Entrega:** Envie o arquivo `.pbix` / `.twbx` exportado, ou publique na versão gratuita da ferramenta e envie o link público.

## 🧠 Dicas
- Menos é mais: não polua a tela com 15 gráficos. Concentre-se em alinhamento, cores consistentes e clareza.
