# 🟡 Nível 03: O Cofre do Pirata (Júnior)

## 📜 O Desafio
Um pirata encontrou um tesouro, mas seu navio suporta apenas um **Peso Máximo**. O tesouro é composto por vários itens, cada um com um `Nome`, um `Valor (em moedas de ouro)` e um `Peso (em kg)`.

Sua tarefa é criar um sistema (pode ser console) que:
1. Permita **Cadastrar** itens disponíveis no tesouro.
2. Permita **Listar** os itens cadastrados.
3. Dada a capacidade máxima de peso suportada pelo navio, implemente um algoritmo que **calcule quais itens o pirata deve levar para maximizar o seu lucro** (valor), sem afundar o navio (ultrapassar o peso).

## 🎯 Objetivo
Avaliar modelagem e Programação Orientada a Objetos, uso de estruturas de dados e implementação de algoritmos de otimização básicos (estilo *Knapsack problem* simplificado ou solução gulosa/greedy).

## 🛠️ Regras
- **Linguagem:** Livre.
- Não é necessário persistir os dados em banco de dados, pode usar memória (listas, arrays).
- Crie uma estrutura limpa e clara. Separar entrada do usuário da lógica de negócio é muito bem-vindo.

## 🧠 Dicas
- Para a lógica do navio, pense na relação `Valor / Peso` de cada item ou pesquise sobre o problema da mochila. O mais importante é o seu raciocínio estruturado no código.
