# 🚀 Sandbox de Entrevistas Técnicas

Bem-vindo ao repositório de Desafios Técnicos! 

Este espaço foi criado para avaliar e explorar a capacidade analítica, raciocínio lógico, criatividade e qualidade de código de nossos candidatos. Aqui, valorizamos muito mais a **forma como você pensa e resolve problemas** do que o conhecimento decorado de uma linguagem específica.

Os desafios estão organizados em **11 níveis (0 a 10)**, divididos pelas categorias de senioridade.

## 🎯 Trilhas e Categorias

Este repositório está dividido em duas grandes trilhas: **Engenharia de Software (Desenvolvimento)** e **Análise de Dados**.
Os desafios estão organizados em níveis de **0 a 10** para ambas as trilhas, de acordo com a senioridade.

### 💻 Trilha 1: Engenharia de Software

#### 🟢 Estagiário (Níveis 0 ao 2)
- **Foco:** Lógica de programação pura, algoritmos fundamentais e criatividade.
- **Linguagem:** **Livre**. Você pode usar a linguagem com a qual se sentir mais confortável.

#### 🟡 Júnior (Níveis 3 ao 5)
- **Foco:** Estruturação de código, Programação Orientada a Objetos, boas práticas e manipulação de estruturas de dados.
- **Linguagem:** **Livre**. Escolha a ferramenta ideal para mostrar sua capacidade de organizar e estruturar código limpo.

#### 🟠 Pleno (Níveis 6 ao 8)
- **Foco:** Desenvolvimento de APIs, integrações, componentização frontend e boas práticas arquiteturais.
- **Stack Obrigatória:** **Java (Backend)** e/se **Angular (Frontend)**. É esperado o uso de frameworks modernos (como Spring Boot).

#### 🔴 Sênior (Níveis 9 ao 10)
- **Foco:** Arquiteturas complexas, resiliência, escalabilidade, segurança, mensageria e gestão de estado avançada.
- **Stack Obrigatória:** **Java (Backend)** e **Angular (Frontend)**. Espera-se domínio profundo de design de sistemas e qualidade técnica.

---

### 📊 Trilha 2: Análise de Dados

#### 🟢 Estagiário (Níveis 0 ao 2)
- **Foco:** Extração básica, manipulação de planilhas/CSVs, limpeza inicial de dados e SQL essencial.
- **Linguagem:** Planilhas (Excel/Sheets), SQL básico ou Python (Pandas).

#### 🟡 Júnior (Níveis 3 ao 5)
- **Foco:** Análise Exploratória (EDA), SQL intermediário e criação de Dashboards interativos.
- **Linguagem:** SQL intermediário, Python (Pandas/Matplotlib) e ferramentas de BI (Power BI, Tableau, Metabase).

#### 🟠 Pleno (Níveis 6 ao 8)
- **Foco:** Modelagem de dados, estatística aplicada (Testes A/B) e Pipelines (ETL) simples.
- **Linguagem:** SQL avançado, Python (Pandas/PySpark). Conhecimento de modelagem dimensional.

#### 🔴 Sênior (Níveis 9 ao 10)
- **Foco:** Arquitetura de Analytics, otimização de performance e modelagem preditiva avançada.
- **Linguagem:** SQL otimizado para Big Data, Python (Machine Learning), arquitetura em cloud.

## 🛠️ Como resolver os desafios

1. **Escolha o Nível:** O recrutador indicará qual pasta/nível você deve resolver de acordo com a vaga pretendida.
2. **Leia as Instruções:** Cada pasta de nível contém o seu próprio `README.md` com a descrição do desafio, regras e critérios de avaliação.
3. **Mão na Massa:**
   - Faça um fork ou clone deste repositório.
   - Crie uma branch para o seu desenvolvimento: `git checkout -b desafio/seu-nome-nivel-x`.
   - Desenvolva sua solução dentro da pasta correspondente ao nível.
4. **Documente:** Se você escolheu uma linguagem livre (Estagiário/Júnior), crie um arquivo com instruções claras de **como compilar e rodar** a sua aplicação.
5. **Entregue:** Faça o commit do seu código, envie para o repositório remoto e abra um Pull Request (ou envie o link do repositório/arquivo zip, conforme alinhado).

---

## 🧠 O que avaliamos?

- **Raciocínio Lógico:** A solução resolve o problema de forma correta e eficiente?
- **Qualidade do Código:** O código é limpo, legível e organizado (Clean Code)?
- **Testabilidade:** Você se preocupou em escrever testes automatizados? (Sempre um diferencial excelente, obrigatório a partir de Pleno).
- **Tratamento de Erros:** O sistema quebra facilmente ou possui boa resiliência contra inputs inesperados?
- **Criatividade:** Em problemas abertos, como você pensou fora da caixa para entregar valor extra?

Boa sorte e divirta-se codando! 🚀
