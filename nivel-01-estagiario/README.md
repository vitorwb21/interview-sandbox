# 🟢 Nível 01: A Máquina de Troco Perfeita (Estagiário)

## 📜 O Desafio
Você foi contratado para criar a lógica de um caixa de supermercado inteligente. O programa deve receber o **Valor Total da Compra** e o **Valor Entregue pelo Cliente**. 

O seu algoritmo deve calcular o troco e informar **qual é a menor quantidade possível de notas e moedas** que devem ser devolvidas ao cliente.

Considere as seguintes notas e moedas (em Reais):
- Notas: 100, 50, 20, 10, 5, 2
- Moedas: 1, 0.50, 0.25, 0.10, 0.05, 0.01

## 🎯 Objetivo
Avaliar o raciocínio matemático e a capacidade de utilizar laços de repetição e condicional para resolver um problema comum e prático.

## 🛠️ Regras
- **Linguagem:** Livre.
- A solução deve lidar com **casos excepcionais**, por exemplo:
  - O valor entregue é menor que o valor da compra.
  - O valor entregue é exatamente igual ao da compra.
- A saída deve mostrar a quantidade exata de cada nota/moeda a ser devolvida.

## 🧠 Dicas
- Pense na ordem de divisão para obter sempre as maiores notas primeiro.
- Cuidado com o arredondamento de casas decimais em linguagens de programação!
