# 🟡 Dados Nível 03: Análise Exploratória (EDA) Visual (Júnior)

## 📜 O Desafio
A equipe de Marketing quer entender o perfil dos clientes da nossa plataforma de streaming. Você recebeu um dataset contendo: `idade`, `genero_preferido`, `horas_assistidas_por_mes` e `plano_assinatura` (Basic, Standard, Premium).

Você deve realizar uma **Análise Exploratória de Dados (EDA)** que responda graficamente:
1. Qual a distribuição de idades dos nossos clientes?
2. Existe relação entre o plano de assinatura e as horas assistidas?
3. Qual é o gênero preferido por faixa etária (ex: 18-25, 26-35, etc.)?

## 🎯 Objetivo
Avaliar o domínio de bibliotecas de visualização e a capacidade de traduzir números em insights visuais.

## 🛠️ Regras
- **Ferramenta Recomendada:** Python (Pandas + Matplotlib, Seaborn ou Plotly). Jupyter Notebook é a melhor forma de entregar este desafio.
- Os gráficos devem ter títulos, eixos nomeados e legendas claras.

## 🧠 Dicas
- Um bom analista não entrega apenas o gráfico, ele escreve uma ou duas frases de conclusão embaixo da imagem. Ex: "Nota-se que clientes Premium assistem 30% mais horas."
