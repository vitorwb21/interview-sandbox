# 🟢 Dados Nível 02: O Resumo Estatístico (Estagiário)

## 📜 O Desafio
Em posse de um dataset público (ex: "Kaggle Titanic Dataset" ou "Iris Dataset", que você deverá baixar), sua missão é extrair os indicadores básicos que todo analista de dados deve saber de cor.

Calcule para a coluna de uma variável contínua (ex: idade ou preço):
1. A média.
2. A mediana.
3. A moda (se aplicável).
4. O valor máximo e o valor mínimo.

## 🎯 Objetivo
Verificar se o candidato entende os conceitos estatísticos descritivos mais fundamentais e sabe aplicá-los programaticamente.

## 🛠️ Regras
- **Ferramenta:** Python (Pandas/Numpy), R, ou até SQL se preferir importar o CSV.
- Exiba os resultados de forma amigável no console ou em um arquivo markdown.

## 🧠 Dicas
- Por que a mediana às vezes é melhor do que a média? (Tente responder essa pergunta no seu documento final!).
